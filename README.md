## 📌 Sobre el Proyecto

Este repositorio contiene el código fuente de mi portfolio profesional. El sitio fue desarrollado desde cero con un enfoque fuerte en el diseño UI/UX, utilizando una estética minimalista y cálida inspirada en textiles crudos e hilos (paleta de colores: piedra, lino, verde salvia y terracota).

El objetivo principal de esta plataforma es exhibir mis proyectos de rebranding, diseño de logotipos, serigrafía y desarrollo de software, demostrando control absoluto sobre la maquetación web y la lógica del lado del cliente.

## ✨ Características Principales

*   **Maquetación Moderna y Fluida:** Uso avanzado de **CSS Grid** y **Flexbox** para lograr un diseño 100% responsivo y adaptable a cualquier dispositivo, utilizando unidades relativas (`rem`, `vw`, `vh`) para una escalabilidad perfecta.
*   **Gestión Dinámica del DOM:** La sección de proyectos destacados se renderiza e inyecta dinámicamente utilizando JavaScript (Vanilla JS).
*   **Iconografía Vectorial (SVG):** Implementación de íconos SVG inline personalizados y estilizados directamente desde CSS mediante variables, garantizando nitidez sin carga de peticiones externas.
*   **Formulario de Contacto Seguro:** 
    *   Validación asíncrona de datos en el cliente mediante JavaScript.
    *   Integración con la API de **Formspree** usando `fetch()`.
    *   Protección contra spam y bots mediante técnica *Honeypot* oculta en el DOM.
*   **Accesibilidad y Performance:** Soporte nativo para la preferencia del usuario de reducción de movimiento (`prefers-reduced-motion`) y optimización de jerarquías de etiquetas semánticas.

## 🛠️ Tecnologías Utilizadas

*   **Frontend:** HTML5 Semántico, CSS3 (Variables, Custom Properties, Media Queries).
*   **Lógica:** JavaScript Moderno (ES6+, Promesas, Async/Await).
*   **Tipografía:** Fraunces (Display), Work Sans (Cuerpo), Space Mono (Detalles técnicos).
*   **Backend / Emailing:** Formspree (BaaS para recepción de datos).
